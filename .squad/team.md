### Capabilities

**🟢 Good fit — auto-route when enabled:**
- Bug fixes with clear reproduction steps
- Test coverage additions
- Lint, formatting, and cleanup work
- Small isolated features with clear acceptance criteria
- Boilerplate and scaffolding
- Documentation fixes

**🟡 Needs review — route to @copilot but require squad review:**
- Medium features with established patterns
- Refactors with solid test coverage
- Endpoint or integration work following existing contracts

**🔴 Not suitable — route to squad member instead:**
- Architecture decisions and system design
- Accessibility-critical UX decisions
- Security-sensitive or performance-critical work
- Ambiguous work needing product judgment
- Cross-team coordination
